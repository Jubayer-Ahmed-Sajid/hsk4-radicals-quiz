How to Use This Deck
1. First Pass (Exposure): Spend 3 seconds per card. Read the character, pinyin, and meaning.
Don't try to memorise -- just expose.
2. Second Pass (Recognition): Cover the meaning. Can you recall it from the character
alone? Tick the checkbox if yes.
3. Third Pass (Production): Use the Self-Test section.Read the English, write the character
and pinyin from memory.
4.Review Cycle:Revisit unticked cards daily until all boxes are checked.
